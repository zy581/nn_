# 车道与路径检测项目
**作者**：ultra223  
**进度**：1/10（基础模块完成）

## 项目简介
基于 OpenCV 实现自动驾驶场景下的车道线检测与路径规划算法，包含图像预处理、边缘提取、感兴趣区域提取、车道线拟合、路径生成等完整功能模块。

## 当前完成内容（1/10）
1. 图像读取与异常处理
2. 灰度化与高斯模糊降噪
3. Canny 边缘检测
4. 车道感兴趣区域（ROI）自动提取
5. 多窗口结果可视化展示
6. 完整代码结构与规范注释

## 开发环境
- Python 3.8+
- OpenCV-Python
- NumPy
