# 图像目标检测系统

## 项目概述

本项目基于深度学习目标检测算法，在 [OpenHUTB/nn](https://github.com/OpenHUTB/nn.git) 开源仓库基础上进行迭代开发，构建了一套**多功能、高性能的图像目标检测系统**。系统集成**单图像检测、批量图像检测、视频检测、摄像头实时检测**等全场景能力，同时提供**检测结果统计分析、多模型对比测试、结果保存与导出**等增强功能，为自动驾驶环境感知、智能监控等场景提供完整的解决方案。

## 相关算法与理论基础

### 1. YOLOv8 目标检测算法

项目采用 YOLOv8 作为核心检测模型，其核心架构包含：

- **骨干网络（Backbone）**：采用 CSPDarknet 结构，通过跨阶段局部网络（CSP）实现高效的特征提取
- **颈部网络（Neck）**：采用 PAN（Path Aggregation Network）结构，实现多尺度特征融合
- **头部网络（Head）**：采用解耦头设计，分别处理分类和回归任务

YOLOv8 的核心优势包括：

- 更高的检测精度与速度平衡
- 支持多种模型尺寸（n/s/m/l/x）
- 原生支持 PyTorch 生态

### 2. 单目视觉距离估算

基于相机成像原理，利用目标检测框尺寸估算目标距离：

\[
Z = \frac{H \times f}{h}
]

其中：

- Z：目标与相机的实际距离（m）
- H：目标真实物理高度（m）
- f：相机等效焦距（像素）
- h：目标检测框高度（像素）

### 3. 危险等级评估

根据距离阈值划分危险等级：

\[
\text{危险等级} =
\begin{cases}
\text{DANGER}, & Z < 10m \\
\text{WARNING}, & 10m \leq Z < 20m \\
\text{SAFE}, & Z \geq 20m
\end{cases}
]

## 功能模块

### 1. 核心检测引擎

`detection_engine.py` 封装 YOLOv8 模型加载与推理逻辑，提供：

- 模型静默加载（抑制冗余输出）
- 置信度阈值配置
- 距离估算与危险等级标注
- 检测结果可视化

### 2. 批量检测模块

`batch_detector.py` 支持批量处理图像文件夹：

- 自动遍历支持的图像格式（jpg、png、bmp等）
- 并行处理与进度追踪
- 结果统一保存到输出目录

### 3. 多模型对比测试

`model_comparison.py` 实现多模型对比分析：

- 同时加载多个 YOLOv8 模型
- 单图像/批量图像对比测试
- 推理速度基准测试
- 检测性能统计报告生成

### 4. 统计分析模块

`stats_analyzer.py` 提供检测结果深度分析：

- 类别分布统计
- 置信度分布分析
- 空间分布统计
- 报告生成与导出

### 5. 用户交互界面

`ui_handler.py` 提供友好的命令行交互：

- 菜单式功能选择
- 命令行参数支持
- 实时检测预览
- 配置文件管理

## 功能实现与开发详情

### 1. 多模型对比测试功能

新增 `model_comparison.py` 模块，核心实现包括：

- **模型管理器**：支持同时加载多个模型，自动处理加载失败情况
- **对比测试接口**：`compare_on_image()` 和 `compare_on_batch()` 方法实现单图和批量对比
- **基准测试**：`benchmark_models()` 方法支持多次运行取平均，输出统计数据
- **报告生成**：自动生成格式化对比报告，包含检测性能、类别分布、置信度分布等

### 2. 检测结果统计分析

`stats_analyzer.py` 实现全面的统计分析：

- **类别分布**：统计各目标类别的检测数量与占比
- **置信度统计**：计算均值、最大值、最小值、中位数
- **空间分布**：分析目标尺寸与位置分布
- **多格式导出**：支持文本报告和 JSON 格式导出

### 3. 配置文件管理

`config.py` 实现灵活的配置管理：

- 支持外部 YAML 配置文件
- 内置默认配置参数
- 配置验证与错误处理

## 系统特性

1. **多场景支持**：覆盖静态图片、批量图片、视频文件、摄像头实时检测
2. **多模型对比**：支持同时加载多个 YOLOv8 模型进行性能对比
3. **智能分析**：内置统计分析模块，提供深度检测结果分析
4. **灵活配置**：支持配置文件和命令行参数双重配置方式

6. **结果导出**：支持多种格式的结果导出，便于后续分析

## 应用场景

- **自动驾驶环境感知**：实时检测道路目标，提供距离与危险等级信息
