# 基于 PPO 算法的 CARLA 自动驾驶智能体训练

## 项目简介
本项目旨在利用强化学习技术，在本地的 CARLA 仿真环境中训练一辆能够自动驾驶的汽车。我们会使用目前最成熟的 Stable-Baselines3 强化学习库，让车辆像婴儿学步一样，通过不断试错，学会如何控制油门、刹车和方向盘，最终实现平稳驾驶和避障。

## 硬件与开发环境
* **操作系统:** Windows 11
* **显卡环境:** 笔记本 RTX 4060 (本地直接渲染与训练)
* **仿真器版本:** CARLA 0.9.x
* **核心依赖:** Python 3.8, PyTorch, Stable-Baselines3

## 功能研发路线图 (分期提交计划)

### 第一阶段：环境搭建与小车“点火”
* 编写 Python 脚本，将复杂的 CARLA 引擎封装成标准的 Gym 训练环境。
* 接入 PPO 神经网络算法，让小车能够随机做出动作，跑通整个“感知-决策-控制”的闭环代码。

### 第二阶段：模型训练与效果监控
* 开始正式训练，并接入 Tensorboard 数据面板，实时记录小车的训练得分（Loss 和 Reward曲线）。
* 编写测试脚本，让训练好的小车在地图里跑，并保存它的第一人称摄像头画面。

### 第三阶段：多传感器融合避障
* 在单一摄像头的基础上，加入 CARLA 的碰撞传感器。
* 当小车撞到行人或墙壁时给予严重扣分，逼迫神经网络学会如何刹车和变道避让。

### 第四阶段：自动化评估与恶劣天气测试
* 编写一键测试脚本，让小车在随机的 5 个出生点启动，自动计算它到达终点的成功率。
* 加入下雨天和黑夜的天气代码，测试我们训练出的 AI 是不是在晚上也能正常开车。
