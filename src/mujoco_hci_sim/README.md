mujoco_hci_sim 是一款基于 MuJoCo 物理引擎开发的人机交互（HCI）仿真工具，专注于实现肌肉驱动生物力学模型的交互任务建模与强化学习训练。
### 核心功能
- 构建具备感知能力（如自我中心视觉、本体感觉）的生物力学用户模型
- 定义多样化人机交互任务，通过强化学习算法训练模型完成交互目标
- 支持模块化扩展，可灵活新增生物力学模型、感知模块及交互任务
### 核心特性
- 兼容 OpenAI Gym 接口，可直接对接主流强化学习库
- 生成的仿真包可独立共享，适用于人机交互算法验证、生物力学仿真及相关研究场景
- 依赖 MuJoCo 物理引擎及强化学习框架，支持日志记录与仿真过程可视化
- 提供预定义交互任务配置，可快速启动仿真与训练
