# ============================================================
# 安装说明
# ============================================================
#
# 1. 安装 Python 依赖：
#    pip install -r requirements.txt
#
# 2. CARLA 仿真器安装（重要）：
#    - 下载 CARLA：https://github.com/carla-simulator/carla/releases
#    - 解压到指定目录
#    - 启动 CARLA：./CarlaUE4.sh 或 CarlaUE4.exe
#    - 将 CARLA Python API 添加到 Python 路径：
#      export PYTHONPATH=$PYTHONPATH:/path/to/CARLA/PythonAPI/carla/dist/carla-*.egg
#      （Windows：set PYTHONPATH=C:\path\to\CARLA\PythonAPI\carla\dist\carla-*.egg）
#
# 3. 运行项目：
#    - 静态障碍场景：python carla_da_static.py
#    - 动态障碍场景：python carla_da_dynamic.py
#    - 数据采集：     python util/data_collector.py
#
# ============================================================
# 环境验证
# ============================================================
#
# 验证 CARLA 连接（需先启动仿真器）：
#   python -c "import carla; client = carla.Client('localhost', 2000); client.set_timeout(2); print('CARLA OK')"
#
# 验证 OpenCV：
#   python -c "import cv2; print(f'OpenCV: {cv2.__version__}')"

# 验证 Pygame：

# 常见问题

# Q: CARLA 连接超时
# A: 确保 CARLA 仿真器已启动，检查端口 2000 是否可用
#
# Q: 摄像头无画面
# A: 检查 CARLA 版本与 Python API 版本是否匹配
#
# Q: 纯跟踪算法转向抖动
# A: 调整 pure_pursuit 函数中的 steer 死区阈值
