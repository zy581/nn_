基于CARLA模拟器的自动驾驶可视化监控项目

本项目基于CARLA模拟器搭建，聚焦自动驾驶系统的实时监控与可视化需求，整合多维度感知数据、车辆运行状态及路径规划信息，构建全面、直观、低延迟的监控体系。项目核心目标是为自动驾驶算法调试、功能验证及场景复现提供高效支撑，助力提升自动驾驶系统的安全性与可靠性，适用于自动驾驶算法研发、场景测试及教学演示等场景。

项目围绕自动驾驶全流程监控需求，设计六大核心功能模块，实现从环境感知、障碍物识别到车辆状态、路径规划的全维度可视化呈现，具体如下：
1. 多视角摄像头实时采集与展示
整合车辆前视、后视、左视、右视四大核心视角，实时采集模拟器内行驶画面并同步展示，精准还原真实道路行驶环境。采用低延迟渲染技术，清晰呈现道路标线、周边车辆、行人及障碍物等细节；支持画面缩放、暂停、回放操作，便于场景复盘、问题定位及算法效果追溯。
2. 激光雷达BEV鸟瞰图可视化
基于激光雷达采集的点云数据，实时生成BEV（Bird's Eye View）鸟瞰环境模型，将三维点云数据转化为直观的二维俯视图。通过不同颜色区分车辆、行人、道路、障碍物等不同目标，清晰呈现本车与周边环境的相对位置及距离关系，有效弥补摄像头视觉盲区，为环境感知补全及行驶安全判断提供支撑。
3. 障碍物实时检测与精准标注
结合深度学习目标检测算法，实时识别行驶场景中的车辆、行人、骑车人三类核心障碍物。采用边界框精准标注障碍物位置，同步显示障碍物运动方向、行驶速度等关键信息；通过不同颜色及样式区分障碍物类型，当障碍物进入危险距离时自动触发警示提示，及时捕捉潜在安全风险，助力算法调试与安全验证。
4. 车辆核心状态实时监控
实时采集自动驾驶车辆的核心运行参数，包括行驶速度（单位：km/h）、转向角、油门开度、刹车力度等，以“数值+进度条”的形式可视化展示，参数实时同步更新，便于直观判断车辆操控合理性。支持核心参数历史数据回溯，为自动驾驶控制算法的优化提供精准的数据支撑。
5. 自动驾驶路径规划轨迹可视化
实时渲染自动驾驶系统的规划路径，采用动态线条区分当前规划轨迹与历史行驶轨迹，标注路径节点及车辆行驶方向；按路径类型（如规划路径、实际行驶路径）区分颜色，同步显示路径各节点的预期行驶速度等辅助信息，直观呈现自动驾驶决策逻辑，便于判断路径规划的合理性及发现轨迹异常。
6. 交通信号灯状态实时识别与展示
通过图像识别算法实时捕捉场景中的交通信号灯，精准识别红、绿、黄三种核心状态；在监控界面显著位置展示信号灯状态，并标注信号灯剩余时长（若模拟器支持）；当信号灯状态发生变化时，及时更新显示并触发提示，确保自动驾驶系统准确响应交通规则，同时便于核查系统对交通信号的执行情况。
