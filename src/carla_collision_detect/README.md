# CARLA模拟器中的物体与碰撞检测
## 项目简介
本项目旨在通过 Python 脚本与 CARLA 仿真环境进行交互，实现对仿真世界中物体的检测以及车辆碰撞事件的监听与处理。本项目是理解自动驾驶传感器数据流和环境感知的核心基础。

## 开发环境

* **操作系统:** Windows 11
* **仿真器版本:** CARLA 0.9.15
* **Python 版本:** Python 3.10.11
* **IDE:** Visual Studio Code

## 功能规划


### 1. 环境构建与自动部署
* **动态场景生成:** 编写脚本在地图中随机或指定位置生成测试车辆 (Ego Vehicle) 及大量背景交通流 (NPC)。
* **同步模式配置:** 设置仿真世界为同步模式 (Synchronous Mode)，确保 Python 客户端与服务器端的帧率同步，保证检测数据的实时性。

### 2. 多维碰撞感知系统
* **碰撞传感器集成:** 挂载 `sensor.other.collision`，获取碰撞发生时的冲量大小、碰撞点坐标及对方物体 ID。
* **碰撞类型分析:** 对碰撞对象进行分类处理（如：车辆碰撞、撞击路灯/护栏、碰撞行人），并记录日志。
* **入侵检测 (Lane Invasion):** 扩展挂载车道线入侵传感器，监测车辆是否非正常偏离航道。

### 3. 物体检测与测距 (Object Detection)
* **基于语义分割的物体识别:** 利用语义分割摄像头获取环境掩码，通过 OpenCV 处理识别出前方的车辆、行人和交通标志。
* **3D 边界框 (Bounding Box) 绘制:** 在仿真窗口或独立窗口中，实时为检测到的动态物体绘制 3D 边界框。
* **多传感器融合测距:** 结合激光雷达 (Lidar) 和毫米波雷达 (Radar) 数据，实时计算目标物体与本车的相对距离及相对速度。

### 4. 主动安全逻辑控制
* **TTC (Time to Collision) 计算:** 基于检测到的物体距离和速度，实时计算“预期碰撞时间”。
* **紧急制动策略 (AEB):** 当 TTC 低于安全阈值时，通过 Python 脚本自动介入控制车辆刹车，防止碰撞发生。
* **可视化监控面板:** 利用 Matplotlib 或 OpenCV 实时绘制车辆状态曲线（如速度、加速度、距离变化等）。
