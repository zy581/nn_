# ============================================================
# 安装说明
# ============================================================
#
# 1. 基础安装（PyPI 依赖）：
#    pip install -r requirements.txt
#
# 2. CARLA 安装（重要）：
#    - 下载 CARLA 仿真器：https://github.com/carla-simulator/carla/releases
#    - 推荐版本：CARLA 0.9.13 或更高版本
#    - 将 CARLA Python API 添加到 Python 路径：
#      export PYTHONPATH=$PYTHONPATH:/path/to/CARLA/PythonAPI/carla/dist/carla-*.egg
#
# 3. 运行项目：
#    - 启动 CARLA 仿真器：./CarlaUE4.sh 或 CarlaUE4.exe
#    - 运行主程序：python main.py
#
# 4. 项目模块说明：
#    - main.py       : 主程序入口，车辆控制逻辑
#    - config.py     : 系统配置文件
#    - drawer.py     : Pygame 可视化绘制模块
#    - mpc.py        : MPC 模型预测控制器
#    - utils.py      : 坐标转换等工具函数
#
