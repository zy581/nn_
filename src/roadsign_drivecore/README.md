# roadsign_drivecore基于CARLA的交通标志检测与智能车辆控制系统

---

## 一、项目概述
roadsign_drivecore是基于CARLA仿真器搭建的智能驾驶仿真项目，核心集成了YOLOv8交通标志检测、车辆手动/自动驾驶双模式控制、红绿灯识别响应、碰撞安全防护等能力，以模块化增量迭代的方式完成开发，完整呈现智能驾驶系统从基础框架到功能完善的落地过程。
---

## 二、版本迭代与功能清单
# CARLA自动驾驶仿真系统 版本迭代全量功能说明
### V1.0 仿真平台基础底座搭建
完成CARLA仿真环境初始化与车辆Actor动态生成，搭建车载前视相机画面实时渲染管线，实现全局视角与车载第一视角的平滑跟随切换，完成项目最小可运行闭环（MVP）的底层架构搭建。
### V2.0 线控底盘手动驾驶控制系统
打通底层车辆动力学控制接口，实现基于WASD按键的线控转向、驱动扭矩闭环控制，完成空格键触发的液压制动系统紧急制动能力开发，实现全维度基础手动驾驶人机交互闭环。
### V3.0 交通信号感知与规控响应系统
基于计算机视觉实现交通信号灯状态实时检测与语义分类，搭建规控层信号优先级决策逻辑，红灯状态下触发车辆平稳制动至静止，绿灯状态下自动恢复驱动扭矩输出，完成基础交通规则合规性适配。
### V4.0 功能安全冗余与碰撞应急防护系统
新增车身碰撞传感器数据采集与解析模块，实时监测车辆碰撞加速度与接触状态，碰撞触发后立即切断驱动扭矩、锁止制动系统并重置车辆控制权限，实现碰撞后的车辆姿态稳定与失控防护，完善整车功能安全冗余设计。
### V5.0 手动驾驶全场景操控能力拓展
完善车辆动力学反向驱动控制逻辑，新增S键触发的倒车工况扭矩输出与转向联动控制，补全前进、倒车、转向、制动全工况手动驾驶能力，实现泊车、窄路会车等复杂场景的手动操控全覆盖。
### V6.0 交通标志识别与速度合规管控系统
基于视觉感知算法实现道路限速交通标志的实时检测、字符识别与语义解析，搭建整车速度闭环限制策略，自动将车辆最高行驶速度约束在标志限定的合规阈值内，实现道路限速规则的自动化适配。
### V7.0 自适应巡航（ACC）与前车跟驰控制系统
搭建多传感器融合的前车目标检测与距离测算模块，基于安全车距模型设定默认10米最小跟驰安全距离，实现前车速度与相对距离的实时拟合，自动完成驱动/制动的闭环控制，实现全速域自适应跟车巡航能力。
### V8.0 HMI人机交互驾驶状态告警反馈系统
优化人机交互界面（HMI）的驾驶状态可视化能力，手动驾驶模式下触发紧急制动工况时，界面左下角同步触发高优先级红色闪烁告警提示，制动动作解除后告警自动复位，显著提升紧急操作的人机交互反馈感知度。
### V9.0 车道保持辅助（LKA）系统
基于前视相机画面实现车道线语义分割与车道模型实时构建，通过车辆运动学模型计算横向偏移量与航向角偏差，输出方向盘转角闭环控制指令，实现自动驾驶模式下的车道居中保持控制，完成横向控制核心能力搭建。
### V10.0 行车数据采集与离线分析闭环系统
搭建整车全维度行车数据采集总线，实时同步采集车辆行驶速度、驾驶模式、交通信号状态、控制指令等核心运行数据，配置1Hz固定频率的本地落盘存储能力，输出标准化结构化日志文件，为后续算法迭代、场景复现与数据分析提供全量数据支撑。

## 三、运行环境要求
需提前完成以下环境配置，保证项目正常运行：
- 操作系统：Windows 10/11（64位）
- 核心仿真器：CARLA 0.9.15版本
- Python运行环境：Python 3.7.9版本
- 依赖库：`carla`、`pygame`、`numpy`、`ultralytics`（YOLOv8）、`opencv-python`
- 虚拟环境：推荐使用`carla_yolo37_env`虚拟环境运行，适配Python 3.7.9，避免全局环境依赖冲突

---

## 四、运行操作步骤
1.  启动CARLA仿真器：双击打开CARLA安装目录下的`CarlaUE4.exe`，等待地图完全加载至城市主画面
2.  激活项目环境：打开PyCharm，选择已配置好的`carla_yolo37_env`虚拟环境，确认所有依赖库已完成安装
3.  启动项目：打开项目主文件`Main.py`，点击运行按钮，项目启动后默认进入自动驾驶模式

---

## 五、按键操作说明
| 按键/操作 | 对应功能说明 |
| :-------- | :----------- |
| `M`键     | 手动驾驶/自动驾驶模式切换 |
| `W`键     | 手动模式：控制车辆前进 |
| `S`键     | 手动模式：控制车辆倒车 |
| `A`/`D`键 | 手动模式：控制车辆左/右转向 |
| 空格键    | 手动模式：触发车辆紧急刹车 |
| `ESC`键   | 退出项目，自动清理所有仿真资源 |
| 窗口关闭按钮 | 关闭项目窗口，自动清理仿真资源，避免内存泄漏 |

---

## 六、注意事项
1.  运行项目前，必须先启动CARLA仿真器，否则会出现客户端连接失败报错
2.  需保证Python环境、依赖库版本与CARLA仿真器版本适配，避免版本冲突导致程序异常
3.  每次项目运行结束，建议通过`ESC`键或窗口关闭按钮正常退出，确保仿真资源完全释放，避免CARLA内存泄漏
4.  自动驾驶模式下，车辆会自动执行红绿灯、限速规则响应；手动模式下，所有车辆操作需用户手动控制
5.  若出现画面黑屏、程序闪退等问题，可重启CARLA仿真器与项目程序，或检查摄像头相关配置是否正确
