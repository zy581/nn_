# CARLA 0.9.11 中的交通灯检测与自主控制

本项目基于 CARLA 模拟器（0.9.11 版本）实现了一套简易的自主控制系统。该系统利用摄像头数据检测交通信号灯，并相应地控制车辆在红灯时停车、绿灯时行驶。

## 功能特性:

-  **基于 RGB 摄像头和边界框实现交通信号灯检测。
-  **停车与行驶逻辑：车辆在红灯时停驶，绿灯时恢复行驶。
-  **A自主控制：根据交通信号灯状态自动调整油门和刹车。

## 使用的技术

- **模拟器**: CARLA Simulator 0.9.11
- **编程语言**: Python 3
- **依赖库**:
  - `pygame` – 用于渲染仿真窗口
  - `cv2` (OpenCV) – 用于图像处理（扩展功能时使用）
  - `numpy`, `time`, `math`, `random` – 用于数值计算和时序逻辑处理


## 工作原理

1. **传感器配置**: 在 CARLA 中为车辆搭载前置 RGB 摄像头及其他传感器。
2. **交通信号灯检测**:
   - 脚本通过 CARLA 内置的环境信息（边界框）检测交通信号灯。
   -判断信号灯的状态（红灯 / 绿灯）。
3. **控制逻辑**:
   - 若检测到前方有红灯，车辆会施加刹车并停止行驶。
   - 若信号灯为绿灯，或未检测到相关红灯，车辆则继续行驶。
