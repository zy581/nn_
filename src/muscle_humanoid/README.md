# muscle_humanoid
## 项目简介
muscle_humanoid 是基于 **MuJoCo 物理引擎** 与 **Python** 开发的肌肉驱动仿人机器人仿真项目。项目实现了带自我中心视觉感知的生物力学机器人建模、物理模拟、实时可视化，并支持通过强化学习完成人机交互任务训练。

本项目可直接加载肌肉驱动人形机器人模型，运行物理仿真，实时渲染运动状态，输出关节信息、躯干位置与视觉观测数据，为仿生机器人控制、人机交互、强化学习研究提供轻量化、可快速上手的仿真环境。

## 项目结构
```
muscle_humanoid/
├─ humanoid.xml       # MuJoCo 肌肉驱动人形机器人模型文件
├─ main.py            # 主程序：模型加载、仿真运行、可视化
└─ README.md          # 项目说明文档
```

## 快速使用
1. **确保文件路径正确**  
   将 `humanoid.xml` 与 `main.py` 放在同一目录下。

2. **运行仿真**
   ```bash
   python main.py
   ```

3. **程序功能**
   - 加载肌肉驱动人形机器人生物力学模型  
   - 初始化物理状态并设置预设姿势  
   - 打开实时可视化窗口  
   - 输出仿真时间与躯干位置  
   - 自动运行指定时长后结束

## 窗口操作说明
- 左键拖动：旋转视角  
- 右键拖动：平移场景  
- 滚轮：缩放视图  
- Shift + 左键：拖动机器人肢体

## 模型与代码说明
### humanoid.xml
使用 MuJoCo 格式定义肌肉驱动人形机器人，包含：
- 骨骼结构、关节范围、物理属性  
- 肌肉执行器与驱动参数  
- 预设姿势（深蹲、单腿站立、俯卧、仰卧）  
- 碰撞、摩擦、视觉材质配置

### main.py
实现完整仿真流程：
- 加载 XML 模型  
- 初始化仿真数据与姿态  
- 启动可视化窗口  
- 步进式物理仿真  
- 实时状态输出与视图更新

## 扩展与开发
- 切换初始姿势：修改关键帧索引（0~3）  
- 调整机器人物理参数：编辑 XML 中关节、肌肉、执行器  
- 添加控制指令：通过 `data.ctrl` 实现机器人动作控制  
- 接入强化学习：将视觉观测、本体感知作为状态，肌肉控制作为动作  
- 记录数据：保存关节、位置、传感器数据用于训练与分析

## 适用场景
人体生物力学仿真、肌肉驱动机器人控制、自我中心视觉交互、强化学习环境开发、人机交互任务研究。
