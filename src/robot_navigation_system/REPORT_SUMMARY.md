# 机器人自主导航系统 - 项目汇报

## 项目简介

基于深度Q网络（DQN）的机器人自主导航系统，通过激光雷达感知和强化学习实现智能路径规划与避障。

---

## 核心技术

### 技术栈
- **深度学习**: PyTorch 2.11
- **强化学习**: DQN算法
- **环境感知**: 激光雷达（360度扫描）
- **可视化**: Matplotlib

### 网络架构
```
输入(364维) → FC(256) → FC(128) → FC(64) → 输出(5维)
```

1. **环境感知**
   - 激光雷达有效探测障碍物
   - 360度扫描提供全面信息
   - 噪声模拟真实传感器特性

2. **学习效果**
   - 训练50轮后奖励明显提升
   - DQN成功学习避障策略
   - ε-贪心策略平衡探索与利用

3. **导航性能**
   - 机器人能够避开障碍物
   - 路径规划基本合理
   - 目标定位准确
