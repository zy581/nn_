# 湿滑路面附着系数估计与自适应 ABS 制动控制

本项目实现了一个面向智能车辆主动安全的湿滑路面制动控制演示模块。车辆在紧急制动时，如果遇到湿地、薄冰等低附着路面，固定大力制动容易造成车轮抱死，滑移率过高，轮胎无法工作在最佳附着区间，导致制动距离变长和方向稳定性下降。本项目通过构建混合附着路面场景，对比普通抱死制动和自适应 ABS 制动控制，展示滑移率控制与路面附着估计对车辆安全性的提升。

## 功能内容

- 构建包含干燥沥青、湿滑沥青、薄冰、恢复沥青的混合路面。
- 设计 30 m/s 初速度下的紧急制动仿真实验。
- 实现固定大力制动基线策略，模拟车轮高滑移率抱死状态。
- 实现自适应 ABS 控制策略，使滑移率保持在最佳附着区间附近。
- 根据制动减速度和轮胎滑移利用率估计路面附着系数。
- 输出制动距离、制动时间、最大滑移率、平均滑移误差和舒适性代价。
- 生成速度-距离曲线、附着系数估计曲线、滑移率控制响应图和指标对比图。
- 导出 `braking_timeseries.csv` 和 `metrics.json`，方便复现实验和写提交说明。

## 运行方法

在本目录执行：

```bash
python main.py
```

或指定输出目录：

```bash
python main.py --output assets
```

运行后会生成：

- `speed_distance_comparison.png`：抱死制动和自适应 ABS 的速度-距离对比。
- `friction_estimation.png`：真实路面附着系数与估计附着系数曲线。
- `slip_control_response.png`：两种制动策略的车轮滑移率响应。
- `metric_comparison.png`：制动距离、制动时间、最大滑移率和滑移误差对比。
- `braking_timeseries.csv`：两种策略的时序仿真数据。
- `metrics.json`：核心指标和优化百分比。

## 项目意义

该项目体现了智能车辆主动安全中的一个关键问题：车辆不仅要知道“需要刹车”，还要根据当前路面附着能力决定“怎样刹车”。当路面从干燥沥青突然进入薄冰区域时，固定制动策略容易产生高滑移率，而自适应 ABS 通过估计附着系数并控制滑移率，可以让轮胎保持更高的有效抓地能力，从而缩短制动距离并提升稳定性。

该模块还可以继续扩展到四轮独立制动、弯道制动、坡道路面、轮胎温度模型和真实传感器噪声场景。

## 测试

```bash
python -m pytest tests
```

测试内容包括：

- 轮胎抓地能力是否在最佳滑移率附近达到更高值。
- 自适应 ABS 是否缩短制动距离并降低最大滑移率。
- 主程序是否能生成指标文件和可视化图片。
