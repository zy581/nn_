# CARLA 自动驾驶仿真平台
基于 CARLA 的多传感器融合、可视化与安全预警自动驾驶仿真系统，支持四路相机、BEV 鸟瞰图、天气切换、数据记录与碰撞危险检测，适用于教学演示、算法验证。

## ✨ 项目特性
- 🚗 **四路 RGB 相机**：前 / 后 / 左 / 右实时画面显示
- 🌍 **BEV 鸟瞰图可视化**：激光雷达点云 + 车道线 + 可行驶区域
- 🌤 **多环境切换**：白天 / 夜间 / 雨天 / 雾天 一键切换
- ⚠️ **安全预警系统**：近距离逼近、碰撞风险、急刹行为检测
- 📊 **数据记录**：黑匣子记录（位置/速度/时间）+ 场景录制回放
- 🚦 **辅助驾驶**：红绿灯监测 + 虚拟仪表盘
- 🧩 **模块化架构**：代码结构清晰，易于扩展与维护
