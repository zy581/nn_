# CARLA 0.9.15 中的交通灯检测与自主控制
本项目使用 CARLA 仿真器（0.9.15 版本）实现了一套简易自主控制系统。系统利用摄像头数据检测**交通灯**，并控制车辆**红灯停车**、**绿灯通行**。

## 功能特点：
- 利用 RGB 摄像头与边界框进行**交通灯检测**。
- **启停逻辑**：车辆在红灯时停下，绿灯时恢复行驶。
- **自主控制**：根据交通灯状态自动调整油门与刹车。

## 使用技术
- **仿真器**：[CARLA Simulator 0.9.11](https://github.com/carla-simulator/carla)
- **语言**：Python 3
- **库**：
  - `pygame` - 用于渲染仿真窗口
  - `cv2`（OpenCV） - 用于图像处理（如需扩展）
  - `numpy`、`time`、`math`、`random` - 用于计算与时序逻辑

## 工作原理
1.  **传感器配置**：在 CARLA 中为车辆配备前置 RGB 摄像头及其他传感器。
2.  **交通灯检测**：
    - 脚本利用 CARLA 内置环境信息（边界框）检测交通灯。
    - 判断信号灯状态（红灯/绿灯）。
3.  **控制逻辑**：
    - 若前方出现红灯，车辆执行刹车并停下。
    - 若为绿灯或无相关红灯，车辆继续行驶。

## 项目运行方法
1.  **安装 Python 依赖项**：
```bash
pip install pygame opencv-python numpy
```
   
2. **使用方法:**
CarlaUE4.exe    # On Windows

3. **预期行为:**
    车辆在检测到前方红灯时平稳停下。
    信号灯变绿后恢复行驶。
    所有行为由控制逻辑自动处理。
