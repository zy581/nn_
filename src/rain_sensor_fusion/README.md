# rain_sensor_fusion
基于多传感器融合的自动驾驶雨天环境 3D 物体检测仿真项目

## 项目简介
针对雨天雨雾遮挡、镜头沾水、激光雷达点云稀疏、视觉特征退化等问题，融合视觉相机、激光雷达 LiDAR、毫米波雷达多传感器数据，在 CARLA 仿真平台下实现复杂雨天场景的 3D 车辆、行人、障碍物检测模拟，提升自动驾驶感知系统在恶劣天气下的检测精度与鲁棒性。

## 核心功能
1. 构建雨天仿真环境：雨雾、积水、光照变化等恶劣天气模拟；
2. 模拟车载多传感器部署：RGB 相机、激光雷达、毫米波雷达同步采集数据；
3. 多传感器数据配准与融合，弥补单一传感器在雨天环境下的感知缺陷；
4. 实现 3D 目标检测，输出障碍物 3D 框、类别、位置、尺寸、朝向信息；
5. 可视化检测结果、传感器点云、融合效果与雨天感知对比实验。

## 技术栈
- 仿真平台：CARLA
- 编程语言：Python
- 核心依赖：PyTorch、NumPy、OpenCV、Pygame
- 感知算法：多传感器数据融合、3D 点云处理、图像特征提取、3D 目标检测
